## 백준 12605 단어순서뒤집기
sol)문장 전체를 뒤집고, ' '기준으로 단어를 다시 뒤집는다.

```
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
	int n;
	cin >> n;
	cin.ignore();
	for (int i = 1; i <= n; i++) {
		string str;
		getline(cin, str);
		reverse(str.begin(), str.end()); //전체 순서 뒤집기
		str += ' ';

		int front = 0;
		for (int end = 0; end < str.size(); end++) {
			if (str[end] == ' ') {
				reverse(str.begin() + front, str.begin() + end); //단어 순서 뒤집기
				front = end + 1;
			}
		}
		cout << "Case #" << i << ": " << str << "\n";
	}
}

```

## 백준 25593 근무지옥에 빠진 푸앙이
tip) 맵 사용해서 {key, value} 형식으로

```
#include <iostream>
#include <string>
#include <map>
using namespace std;

int main() {
	int n;
	map<string, int> table;
	int time = 0;
	cin >> n;
	for (int i = 0; i < n; i++) { //주
		string str;
		for (int j = 0; j < 4; j++) { //근무 시간대
			if (j == 0 || j == 2) time = 4;
			else if (j == 1) time = 6;
			else if (j == 3) time = 10;
			for (int k = 0; k < 7; k++) { //월화수목금토일
				cin >> str;
				if ((str[0] >= 'a' && str[0] <= 'z')) { //이름이면
					if (table.find(str) != table.end()) // table에 이미 있으면
						table.find(str)->second += time;
					else
						table.insert(make_pair(str, time));
				}
			}
		}

	}
	int flag = 1;
	for (auto iter1 = table.begin(); iter1 != table.end(); iter1++) { //순회
		for (auto iter2 = table.begin(); iter2 != table.end(); iter2++) {
			if (iter1->second - iter2->second < -12 || iter1->second - iter2->second > 12) {
				flag = 0; break;
			}
		}
		if (!flag) break;
	}
	if (!flag) cout << "No\n";
	else cout << "Yes\n";
}

## 백준 25501 재귀의 귀재

```
#include <iostream>
#include <cstring>
using namespace std;

int cnt;

int recursion(const char *s, int l, int r) {
	cnt++;
	if (l >= r) return 1;
	else if (s[l] != s[r]) return 0;
	else return recursion(s, l + 1, r - 1);
}

int isPalindrome(const char *s) {
	return recursion(s, 0, strlen(s) - 1);
}

int main() {
	int t;
	scanf(" %d", &t);
	for (int i = 0; i < t; i++) {
		char s[1001];
		scanf(" %s", &s);
		cnt = 0;
		printf("%d ", isPalindrome(s));
		printf("%d\n", cnt);
	}
}

## 백준 24416 피보나치 수1

```
#include <iostream>
using namespace std;

int recur = 0;
int dp = 0;
int fib(int n) {
	if (n == 1 || n == 2) {
		recur++;
		return 1;
	}
	else return (fib(n - 1) + fib(n - 2));
}
int fibonacci(int n) {
	int f[41];
	f[1] = f[2] = 1;
	for (int i = 3; i <= n; i++) {
		dp++;
		f[i] = f[i - 1] + f[i - 2];
	}
	return f[n];
}

int main() {
	int n;
	cin >> n;
	fib(n);
	fibonacci(n);
	cout << recur << " " << dp << "\n";
}

#include <iostream>
#include <algorithm>
using namespace std;
int main() {
	int r;
	scanf(" %d", &r);
	char *sang = new char[r];
	for (int i = 0; i < r; i++)
		scanf(" %1c", &sang[i]);
	int n;
	scanf(" %d", &n);
	char *friends = new char[r*n];
	for (int i = 0; i < r*n; i++)
		scanf(" %1c", &friends[i]);

	int res = 0;
	int exp_res = 0;
	for (int i = 0; i < r; i++) {//상근이와 친구들을 한 번에(세로로) 비교
		int rock = 0, sci = 0, paper = 0;//친구들
		for (int j = 0; j < n; j++) {// n번 반복
			if (friends[i + j * r] == 'R')
				rock++;
			else if (friends[i + j * r] == 'S')
				sci++;
			else if (friends[i + j * r] == 'P')
				paper++;
		}
		if (sang[i] == 'R')
			res += sci * 2 + rock;
		else if (sang[i] == 'S') 
			res += paper * 2 + sci;
		else if (sang[i] == 'P')
			res += rock * 2 + paper;
		exp_res += max({ sci * 2 + rock,paper * 2 + sci,rock * 2 + paper }); //상근이가 r,s,p일 때의 최대값
		//--------------exp_res를 구할 때 r,s,p의 최대/중간값에 따라 상근이가 내야할 손모양이 무엇인지의 방향으로 풀려 했었다...
	}

## 백준 4796 캠핑
q) 더 빠른 방법?

```
#include <iostream>
using namespace std;
int main() {
	int l, p, v;
	int i = 1;
	while (1) {
		int res = 0;
		cin >> l >> p >> v;
		if (!l && !p && !v) //종료 조건
			break;
		if (v%p > l) //20%8 == 4가 > ㅣ=3인 경우, 최대 3일까지만 가능. 고려
			res = l;
		else
			res = v % p;

		cout << "Case " << i << ": " << (v / p)*l + res << "\n";
		i++;
	}
}
