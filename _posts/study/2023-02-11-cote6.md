## 백준 1010 다리 놓기
###### 오답
~~왜 틀렸는지 모르겠다. 예제 입출력은 맞게 나온다.~~
```c++
//sol) 조합의 기본 성질을 이용한다.
#include <iostream>
using namespace std;

int main() {
	int t;
	cin >> t;
	for (int i = 0; i < t; i++) { //테스트케이스
		int n, m;
		long long res = 0;
		cin >> n >> m; //mCn
		if (n == m)
			res = 1;
		else {
			long long de = 1, nu = 1; //분모, 분자
			for (int j = 1, k = m; j <= n; j++, k--) { //n번 반복
				nu = nu * k;
				de = de * j;
			}
			res = nu / de;
		}
		cout << res << "\n";
	}
}
```
<br>

## 백준 1343 폴리오미노
##### 오답
~~이건 또 왜 틀렸지...char[]로 풀었는데 string으로 풀 걸 그랬나~~
```c++
#include <iostream>
#include <cstring> //memset, strlen
using namespace std;

int main() {
	char board[50];
	cin >> board;

	char res[50];
	memset(res, 'A', sizeof(char) * 50); //A로 초기화
	int cnt = 0, flag = 0; //cnt: '.' 나오기까지의 'X' 개수, flag: -1 or 값 출력
	for (int i = 0; i < strlen(board); i++) {
		if (board[i] == '.') {
			res[i] = board[i];
			if (cnt) { //그 전에 'X'가 나왔던 경우
				if (cnt % 2) {
					flag = 1;
					break;
				}
				else if (cnt % 4) { //4 + 2 조합일 때
					for (int j = i; j > i-2; j--) {
						res[j-1] = 'B';
					}
				}// 4 조합은 초기화를 이미 했기 때문에 할 필요 없음
			}
			cnt = 0;
			continue;
		}
		cnt++;
	}
	if (cnt) { //'.' 이후 'X'개수 or '.'없이 'X'만 있던 경우
		if (cnt % 2)
			flag = 1;
		else if (cnt % 4) { //4 + 2 조합일 때
			for (int j = strlen(board); j > strlen(board) - 2; j--) {
				res[j - 1] = 'B';
			}
		}
	}
	if(flag) //출력
		cout << "-1";
	else {
		for (int i = 0; i < strlen(board); i++)
			cout << res[i];
	}
}
```

** char 배열 초기화

- ```c++
  char arr[50] = {0, }; //arr을 출력하면 아무것도 나오지 않는다. 0이 아닌 NULL의 의미다.
  ```
&nbsp;&nbsp;&nbsp;특정 문자로 초기화하기 위해선 다른 방법을 써야 한다. 위 방법은 0만 된다.
- ```c++
  char arr[50] = {'A', }; //첫 요소만 'A'이고 나머진 0. 쉼표는 생략 가능
  ```
- ```c++
  memset(arr, 'A', sizeof(char) * 50); //전부 'A'로 초기화
  ```
